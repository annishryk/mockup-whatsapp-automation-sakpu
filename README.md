# SAKPU WhatsApp Automation - Interactive Mockup

Mockup interaktif alur WhatsApp Automation untuk **SAKPU (Sistem Aplikasi Kebijakan Persaingan Usaha)** milik KPPU (Komisi Pengawas Persaingan Usaha) Republik Indonesia.

🔗 **Live Demo**: [sakpu-whatsapp-mockup](https://mockup-whatsapp-automation.gc-smaart.sytes.net/)

---

## Tentang Project

Mockup ini mensimulasikan alur percakapan WhatsApp antara pengguna dan bot SAKPU, mulai dari registrasi hingga mendapatkan hasil analisis kebijakan persaingan usaha (DPKPU).

---

## Fitur Mockup

- **Tampilan WhatsApp** — UI mirip WhatsApp asli
- **9 Step Navigasi** — Bisa loncat antar step
- **Quick Buttons** — Klik tombol untuk simulasi respons
- **Input Manual** — Bisa ketik pesan sendiri
- **Progress Bar** — Indikator kemajuan simulasi
- **Reset Journey** — Mulai ulang dari awal

---

## Alur Simulasi

| Step | Nama | Keterangan |
|------|------|------------|
| 1️⃣ | **Registrasi** | Daftar akun SAKPU |
| 2️⃣ | **Menu Utama** | Pilih fitur |
| 3️⃣ | **Pilih Simulasi** | Info DPKPU |
| 4️⃣ | **Identitas** | Konfirmasi data diri |
| 5️⃣ | **Info IPLP** | Input judul & deskripsi kebijakan |
| 6️⃣ | **DPKPU Q1-5** | 5 pertanyaan utama |
| 7️⃣ | **Kelengkapan** | 3 pertanyaan kelengkapan |
| 8️⃣ | **Review** | Ringkasan & pernyataan |
| 9️⃣ | **Hasil Final** | Analisis & rekomendasi |
